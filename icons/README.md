## Design Notes

The Voidium Code logo features:
- **Shape**: Brackets `<>` style representing code
- **Colors**: Purple/indigo gradient (`#6366F1` → `#A855F7`)
- **Style**: Modern, minimal with subtle cosmic elements
- **Insiders**: Cyan accent (`#22D3EE`) to distinguish preview builds
