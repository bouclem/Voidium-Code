## Design Notes

The Voidium Code logo features:
- **Shape**: Brackets `>` style representing code, with a horizontal line and dot
- **Colors**: Purple/indigo gradient (`#6366F1` → `#8B5CF6` → `#A855F7`) for stable
- **Insiders**: Cyan/purple gradient (`#22D3EE` → `#6366F1` → `#A855F7`) with "INSIDERS" label
- **Style**: Modern, minimal with subtle cosmic/void elements (decorative dots)
- **Standalone variants** (`cnl`, `clt`): No background rect, used for Windows installer graphics
- **Border variant** (`w80_b8`): Logo at 80% scale with subtle gradient border, for special contexts

## File Naming Convention

- `voidium` — Main logo with background
- `voidium_light` — Light theme with background
- `voidium_cnl` — Color, No background, Logo only
- `voidium_clt` — Color, Light, Transparent (no background)
- `voidium_cnl_w80_b8` — Color, No background, 80% width, 8pt border
